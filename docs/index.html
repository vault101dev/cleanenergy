This is a live demo of a solar estimator built on a custom Model Context Protocol (MCP) server — the same protocol Claude uses to call external tools. The server exposes 4 tools that chain live NREL and EIA government data into real production and savings estimates. It also runs as a Claude Desktop extension. View source on GitHub →
